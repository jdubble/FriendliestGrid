About the Friendliest Grid

The Friendliest Grid is built on two values simplicity and compatibilty. The grid's options are limited and easy to use for that reason. Whether you are a graphic designer or not, you can see grids used everywhere. On the web, an exremely large portion of responsive websites have a maximum of four columns in each row. This Grid will give you the option to use one, two, three, or four columns for each row. If you like, you may also set a maximum width so the row doesn't fill outwards to the edges of the screen or monitor the user has.

The borders and the background colors are just there for demonstration purposes and can easily be removed in the css file. Simply locate the lines with the word border 'border: solid 1px red;' and delete the whole line including the semicolon ;

To remove the background colors locate the four background color properties in each column. They look like this with a different color for each. 'background-color: orange;'. Simply delete these four lines and you are ready to begin.

How to use the Friendliest Grid

How to set up a four column grid

To create a four column grid use one row with the class 'row fourcol'. Then nest four divs with the class 'col'. Easy Peasy eh? Even if you decide to not put any content in the first, second or third column, it will still keep it's alignment. For example, some headers just have a logo on the far left and a search box on the far right. This would work perfect in the Friendliest Grid because, although you would still have to create the divs for each column you could easily leave them empty and just fill the first and fourth column with content.
