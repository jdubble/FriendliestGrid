About the Friendliest Grid

The Friendliest Grid is built on two values simplicity and compatibility. The grid's options are limited and easy to use for that reason. Whether you are a graphic designer or not, you can see grids used everywhere. On the web, an extremely large portion of responsive websites have a maximum of four columns in each row. This Grid will give you the option to use one, two, three, or four columns for each row. You can also use a special case and span 2 rows in a three column row. This enables another level of control. To begin all columns are nested within a row.

The borders are just there for demonstration purposes.

How to use the Friendliest Grid

Its simple, just begin with a row. Then choose how many columns you desire within the row. You can choose from one to four columns. Also, within a three column row, you can make one column double wide and just add one more column. This is great for menus or any other element you need to span two columns.

How to set up a four column grid

To create a four column grid use one row with the class 'row fourcol'. Then nest four divs with the class 'col'. Easy Peasy eh? Even if you decide to not put any content in the first, second or third column, it will still keep it's alignment. For example, some headers just have a logo on the far left and a search box on the far right. This would work perfect in the Friendliest Grid because, although you would still have to create the divs for each column you could easily leave them empty and just fill the first and fourth column with content.

How to set up a three column grid

To create a three column grid use one row with the class 'row threecol'. Then nest three divs with the class 'col'. Yes that's all you have to do!

How to set up a three column grid where one column spans the length of two columns.

Use the same strategy as creating a three column grid except add the class 'spantwo' to either of the columns. Because you're only using two columns you only need to write two cols, one should be just 'col' and the other should be 'spantwo'. EX. div class="row threecol"

This column spans two columns width to give you more control. Also the text aligning is centered for ease of design and for responsiveness. You may change anything to your specifications. The class for this nested div is still col but we added spantwo as well.
EX. div class="spantwo col"

Here is the third and final column. This div is nested inside a row with the class of "col". Make sure to only add two nested divs altogether because we are spanning two cols with one col so it still counts as two columns.
